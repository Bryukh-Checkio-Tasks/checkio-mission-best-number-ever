C'était la version et le nombre de Sheldon. Mais vous avez les compétences en code nécessaires pour prouver qu'il existe un meilleur nombre, ou bien que Sheldon a raison. Vous pouvez retourner n'importe quel nombre, mais utilisez votre code pour prouver que votre nombre est le meilleur !

Cette mission est simple à résoudre. On vous donne une fonction "checkio" qui retournera n'importe quel nombre (integer ou float).

Écrivons un essai en python qui explique en quoi votre nombre est le meilleur. Publier la solution de base ne rapporte aucun point, le but étant de gagner des points grâce aux votes pour votre essai.

Données : Non.

Résultat : Un nombre (integer, float ou complex).

Exemple : 

isinstance(checkio(), (int, float, complex))

Utilisation : Il en va de votre capacité à écrire en code et en maths.
